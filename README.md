# idhome-travaux.fr

**ID Home — Rénovation & Dépannage**

Site vitrine pour ID Home, entreprise de rénovation et dépannage en France.

## Stack

- Next.js 15 (App Router) + TypeScript + Tailwind CSS
- Hébergement: Hostinger (GitHub auto-deploy)

## Démarrage

```bash
npm install
npm run dev
```
